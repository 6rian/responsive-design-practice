Pure CSS Line Clamp Toggle

  • The read more/less toggle switch below is implemented using only CSS.
  • This is achieved by using a checkbox to remember state, and a sibling selector to add/remove line clamping.
  • Flexbox is needed to reverse the order of the inner wrapper and checkbox since there isn't a previous sibling selector.
